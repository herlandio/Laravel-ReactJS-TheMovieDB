# API Laravel e ReactJS

> Clone o projeto: <br>

```
git clone https://github.com/herlandio/Laravel-ReactJS-TheMovieDB
```

> Para executar a aplicação basta: <br>

```
docker-compose up -d
```
> Acesse o endereço abaixo para ver a aplicação, após executar o comando acima:

```
http:localhost:8000
```
