# API TheMovieDB with Laravel and ReactJS

https://reactthemoviedb.herokuapp.com/

> Clone the project: <br>

```
git clone https://github.com/herlandio/TheMovieDBLaravelReactJS
```

> Laravel: <br>

```
composer install
```
    
- Linux:
        
```
cp .env.example .env
```

- Windows: 

```
copy .env.example .env
```

```
php artisan key:generate
```

```
php artisan serve
```

> ReactJS:

```
yarn install
```
```
yarn start
```
